# 前台版面修正說明

本次依照截圖問題調整前端版面，主要改善 iPad 橫向 / 直向與長輩操作可讀性。

## 修改檔案

```text
stock-radar-frontend/app.js
stock-radar-frontend/style.css
```

## 已修正內容

1. 主功能區改成大按鈕版面
   - iPad 橫向：10 個功能改為 5 欄 x 2 排，不再全部擠在一排。
   - iPad 直向：改為 2 欄大按鈕。
   - 按鈕高度、字體、間距加大，適合長輩點選。

2. 主力籌碼卡片資訊不再黏在一起
   - 「大戶資料累積中」與「收盤 / 漲跌」分成不同區塊。
   - 收盤與漲跌改成獨立膠囊標籤。

3. 法人同步卡片資訊不再黏在一起
   - 「尚無資料」與「收盤 / 漲跌」分成不同區塊。
   - 收盤與漲跌改成獨立膠囊標籤。

4. 產業內法人買超前 3 檔可讀性改善
   - 移除原本整排過亮的藍色底效果。
   - 改成深色卡片，左側顯示 TOP、股票名稱、代號。
   - 右側分成法人買超、分數、漲跌三個欄位。
   - 字體與對比加強，iPad 上比較容易看清楚。

## 驗證重點

1. 手機或 iPad 開啟前台後，功能按鈕應該變成較大的格狀按鈕。
2. 主力籌碼頁面，收盤 / 漲跌不應該和「大戶資料累積中」擠在一起。
3. 法人同步頁面，收盤 / 漲跌不應該和「尚無資料」擠在一起。
4. 產業資金頁面的前 3 檔股票要比原本清楚，不會出現淡字看不清楚。
