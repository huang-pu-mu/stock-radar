# V1.3-3-4：策略回測排行榜前端強化

## 本次目標

強化策略回測頁中的排行榜區塊，讓使用者可以更清楚比較：

- 最佳股票排行
- 最弱股票排行
- 各策略績效排行
- 1 日 / 3 日 / 5 日 / 目前報酬切換
- 正報酬比例、可用樣本數、待資料數

---

## 修改檔案

```text
stock-radar-frontend/app.js
stock-radar-frontend/style.css
stock-radar-frontend/service-worker.js
```

---

## 新增前端功能

### 1. 排行榜模式切換

新增四種模式：

```text
綜合排行
最佳股票
最弱股票
策略排行
```

---

### 2. 排行榜統計卡片

新增：

```text
指定報酬平均
正報酬比例
最佳 / 最弱差距
待資料數
```

---

### 3. 最佳 / 最弱股票卡片強化

每筆排行顯示：

```text
股票代號
股票名稱
來源策略
訊號日
策略分數
結果狀態
1 日報酬
3 日報酬
5 日報酬
目前報酬
進場價
加入自選
看明細
```

---

### 4. 策略績效排行強化

每個策略顯示：

```text
平均報酬
勝率
可用樣本比例
正報酬數
負報酬數
待資料數
```

---

### 5. 修正策略回測頁讀取流程

修正前端 `loadList()` 及 `rerenderCurrentContent()` 中策略回測頁提前 return 的問題，避免進入策略回測頁時只更新標題但沒有真正呼叫資料讀取。

---

