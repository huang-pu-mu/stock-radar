# 收盤價顏色修改說明

本次修改目標：所有前端顯示的「收盤價 / 收盤」依漲跌顯示顏色。

## 修改規則

1. 漲：紅色
2. 跌：綠色
3. 平盤：黑色

## 已修改位置

1. 今日雷達 / 自選股 / 外資排行卡片內的「收盤」文字
2. 股票查詢結果內的「收盤」文字
3. 看明細視窗內的「收盤價」欄位
4. 看明細內股價走勢圖 K 線顏色
5. 看明細內成交量柱狀圖顏色

## 修改檔案

```text
stock-radar-frontend/app.js
stock-radar-frontend/style.css
```

## 注意

平盤文字使用黑色；因目前 APP 是深色背景，所以黑色文字有加淺色底，避免手機上看不清楚。
