# V1.3-3-3 前端策略回測頁

## 完成內容

新增前端「策略回測」頁籤，接上 V1.3-3-2 策略回測 API。

## 新增功能

1. 策略回測頁籤
2. 回測任務 Run ID 選擇
3. 策略篩選
4. 結果篩選：成功、觀察、失敗、待資料
5. 搜尋股票代號、股票名稱、產業、策略名稱、觸發原因
6. 排序：1 日、3 日、5 日、目前報酬、策略分數、訊號日
7. 回測總覽卡片
8. 各策略回測表現
9. 最佳股票排行
10. 最弱股票排行
11. 回測結果卡片
12. 每筆結果顯示 1 日、3 日、5 日、目前報酬
13. 加入自選與看明細

## 同步補強 API

因最新附件內尚未包含 V1.3-3-2 回測 API，本次一併補入：

- GET /strategy-backtests/runs
- GET /strategy-backtests/runs/:runId
- GET /strategy-backtests/results
- GET /strategy-backtests/summary
- GET /strategy-backtests/rankings

## PWA

快取版本更新為：

```text
stock-radar-pwa-v40
```

## 驗收

部署後請檢查：

- 前端是否出現「策略回測」頁籤
- 是否能選擇 Run ID
- 是否能看到回測總覽
- 是否能看到各策略回測表現
- 是否能切換 1 日 / 3 日 / 5 日 / 目前報酬
- 是否能顯示最佳股票排行
- 是否能顯示最弱股票排行
- 是否能篩選策略與結果
- 是否能搜尋股票
- 加入自選是否正常
- 看明細是否正常
