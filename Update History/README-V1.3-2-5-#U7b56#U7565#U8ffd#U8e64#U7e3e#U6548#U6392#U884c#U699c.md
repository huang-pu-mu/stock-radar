# V1.3-2-5：策略追蹤績效排行榜

## 本版目標

把策略追蹤從「單純保存觀察名單」強化成「可以比較策略追蹤效果」的績效排行榜。

## 後端新增 / 強化

### 1. 強化策略追蹤表現欄位

`GET /strategy-watchlist` 與 `GET /strategy-watchlist/performance` 現在會回傳：

- 加入價 `entry_price`
- 加入價日期 `entry_price_date`
- 追蹤後 1 日價格 / 日期 / 報酬
- 追蹤後 3 日價格 / 日期 / 報酬
- 追蹤後 5 日價格 / 日期 / 報酬
- 目前價格
- 目前報酬
- 表現狀態

### 2. 新增績效 API

```http
GET /strategy-watchlist/rankings?metric=current&limit=10
```

支援 metric：

- `current`：目前報酬
- `1d`：1 日報酬
- `3d`：3 日報酬
- `5d`：5 日報酬

回傳內容：

- 整體平均報酬
- 正報酬比例
- 最佳股票排行
- 最弱股票排行
- 各策略績效排行

## 前端新增

策略追蹤頁新增：

- 目前報酬 / 1 日 / 3 日 / 5 日切換
- 平均報酬
- 正報酬比例
- 正 / 負 / 待資料統計
- 目前最佳追蹤
- 最佳股票排行
- 最弱股票排行
- 各策略目前表現
- 每張追蹤卡片顯示 1 日、3 日、5 日、目前報酬

## 判斷邏輯

- 報酬大於 3%：轉強
- 報酬小於 -3%：轉弱
- 介於 -3% 到 3%：觀察中
- 價格不足：等待資料
